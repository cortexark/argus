What Argus tracks

Real-time monitoring of AI agent activity across your system

File Monitoring

Alerts when AI apps read SSH keys, AWS credentials, .env files, browser password stores, or documents outside your project. Uses chokidar watches and macOS Unified Log for real-time detection.

Network Tracking

Logs every outbound connection from AI processes. Identifies known AI service endpoints and flags unknown destinations, suspicious ports, and browser automation via Chrome DevTools Protocol.

Real-time Alerts

Native macOS notifications for credential access, unknown domains, and unusual port activity. Throttled and batched to avoid notification fatigue. Critical alerts include sound.

How it works

  1. Install and launch. Argus starts as a menubar app. No configuration needed — it begins monitoring immediately.
  2. AI agents are detected automatically. Argus identifies Claude, Cursor, Copilot, ChatGPT, Ollama, and 15+ other AI tools by process name.
  3. Activity is logged locally. File reads, network connections, and process events go into a local SQLite database at ~/.argus/data.db. Nothing leaves your machine.
  4. Alerts surface what matters. Credential access, unknown network destinations, and browser automation trigger native notifications so you know immediately.
  5. Review in the dashboard. Click the menubar icon to see a timeline of all AI activity — files accessed, connections made, session history.
